<github チュートリアル>
＜みんなで作業するために必要な手順＞
※（↓↓全体の作業イメージです↓↓）
①git pull で最新の情報をGithubサーバーからローカルに持ってくる
②編集した内容をadd（アップデートしたい内容を追加）する
③addした文をcommit（①の内容をコミット（変更点の説明みたいなイメージ）する）
④git push する（①、②で用意したものを最終的にUpdateする）
ーーーーーーーーーーーーーーー

＜コマンド解説＞
コマンド①git pull
※サーバーから最新の情報をとってきてから作業しましょう

コマンド②git add .
※git add . で編集または作業したファイルの追加を行う
※編集したらとにかく、コマンド②をしましょう
※updateしたい内容をadd（追加）する（イメージ） 

コマンド③git commit -m " 任意の文字列"
#git commit -m "好きなコメント残しましょう☆"
※git commit -m "なんでもよいから文字入れて一文字以上"
※Index.htmlの＜head＞＜/head＞を編集しましたとかコメント残しておくと みんなに分かりやすいゾ☆
※記録に残るので、HTML編集しましたよとか残すのも良き

コマンド④git push
※①～③の内容を最終的にUpdateするコマンド
git push と打ち込むとアップデート完了
ーーーーーーーーーーーーーー
